# Algorithm Basics

An algorithm is a step-by-step procedure to solve a problem.

Different algorithms solving the same problem can have different performance.

---

# Time Complexity

Time complexity describes how the running time grows as the input size increases.

Common complexities:

| Complexity | Meaning |
|---|---|
O(1) | Constant time |
O(log n) | Logarithmic |
O(n) | Linear |
O(n log n) | Efficient sorting |
O(n²) | Quadratic |

---

# Array

Array stores elements in contiguous memory.

| Operation | Complexity |
|---|---|
Access | O(1) |
Search | O(n) |
Insert | O(n) |
Delete | O(n) |
