# CCW (Counter Clock Wise)

## Definition

Determines the orientation of three points.

Given:

- P1
- P2
- P3

Determine whether P3 is:

- Left of directed line P1 → P2
- Right of directed line P1 → P2
- Collinear

---

## Core Idea

Use the cross product.

Vectors:

\[
\overrightarrow{P_1P_2}
\]

\[
\overrightarrow{P_1P_3}
\]

Compute:

\[
\overrightarrow{P_1P_2}
\times
\overrightarrow{P_1P_3}
\]

The sign indicates orientation. :contentReference[oaicite:1]{index=1}

---

## Process

1. Create two vectors
2. Compute cross product
3. Check sign

Result:

- Positive → Left turn (CCW)
- Negative → Right turn (CW)
- Zero → Collinear

---

## Implementation

For points:

\[
A(x_1,y_1)
\]

\[
B(x_2,y_2)
\]

\[
C(x_3,y_3)
\]

Compute:

\[
(x_2-x_1)(y_3-y_1)
-
(y_2-y_1)(x_3-x_1)
\]

---

## Key Insight

> Cross product sign = geometric direction

Everything in Computational Geometry starts from CCW.

### Very Important

CCW is used in:

- Convex Hull
- Segment Intersection
- Polygon Algorithms
- Sweep Line

---

## Complexity

\[
O(1)
\]

# Convex Hull

## Definition

Smallest convex polygon containing all points.

Often described as:

> Stretching a rubber band around all points.

Hull vertices must come from the original point set. :contentReference[oaicite:2]{index=2}

---

## Core Idea

Remove all interior points.

Keep only outer boundary points.

---

## Naive Method

### Process

For every pair of points:

1. Treat pair as candidate edge
2. Check all other points
3. If all points lie on one side:
   - Edge belongs to hull

### Complexity

\[
O(N^3)
\]

# Graham Scan

## Definition

Efficient Convex Hull algorithm using:

- Sorting
- Stack
- CCW

:contentReference[oaicite:4]{index=4}

---

## Core Idea

Sort points by angle.

Maintain a convex boundary using a stack.

Whenever a non-convex turn appears:

→ Remove points until convexity is restored.

---

## Process

### Step 1

Choose base point.

Usually:

- Lowest Y coordinate

### Step 2

Sort remaining points by polar angle.

### Step 3

Push first two points.

### Step 4

For each new point:

Check:

\[
CCW(top-1, top, new)
\]

If convex:

- Push

Otherwise:

- Pop
- Repeat

### Step 5

Stack becomes Convex Hull.
