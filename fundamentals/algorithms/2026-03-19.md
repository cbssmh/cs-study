# AVL Tree

An AVL tree is a self-balancing binary search tree.

It maintains balance by ensuring height differences are small.

Balance Factor:

BF = height(left) - height(right)

Condition:

|BF| ≤ 1

---

# Rotation

Used to restore balance when the tree becomes unbalanced.

## LL Case
Right Rotation

## RR Case
Left Rotation

## LR Case
Left Rotation → Right Rotation

## RL Case
Right Rotation → Left Rotation

# Expression Tree

An expression tree represents arithmetic expressions in a tree structure.

Rules:

- Leaf nodes → numbers
- Internal nodes → operators

Evaluation:

Postorder traversal evaluates the expression.

---

# Infix to Postfix (Shunting Yard)

Converts infix expressions to postfix expressions.

Uses a stack.

Rules:

- Number → output immediately
- Operator → pop higher precedence operators, then push
- "(" → push
- ")" → pop until "("

Example:

Infix:

2 + 3 * 5 + 4

Postfix:

2 3 + 5 * 4 +

---

# Postfix Evaluation

Uses a stack.

Rules:

- Number → push
- Operator → pop 2 values, compute, push result

Example:

2 3 + → 5  
5 5 * → 25  
25 4 + → 29

---

# Recursion

A function that calls itself.

Requires a base condition to stop.

Example:

Factorial

f(n) = n * f(n-1)  
f(0) = 1

---

# Divide and Conquer

Breaks a problem into smaller subproblems.

Each subproblem is solved recursively.

---

# Tower of Hanoi

A classic recursion problem.

Rules:

- Move one disk at a time
- Larger disk cannot be placed on smaller disk

---

# Hanoi Strategy

To move n disks:

1. Move n-1 disks to auxiliary rod
2. Move largest disk
3. Move n-1 disks to target rod

---

# Recurrence Relation

T(n) = 2T(n-1) + 1

---

# Solution

T(n) = 2^n - 1
