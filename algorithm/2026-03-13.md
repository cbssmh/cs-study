# Tree

A tree is a hierarchical data structure.

Key components:

- Root node
- Parent node
- Child node
- Leaf node
- Edge

---

# Binary Tree

A binary tree is a tree where each node has at most two children.

Example:

      A
     / \
    B   C

---

# Binary Search Tree (BST)

Property:

left subtree < parent < right subtree

Example:

      5
     / \
    3   8
   / \   \
  1   4   9

Searching example:

Find 4

5 → left  
3 → right  
4 found

---

# Tree Traversal

Three traversal methods:

Preorder

Root → Left → Right

Inorder

Left → Root → Right

Postorder

Left → Right → Root

---

# Binary Tree Array Representation

If node index = i

Left child

2i + 1

Right child

2i + 2

Parent

(i - 1) / 2

---

# Balanced Tree

Balanced trees maintain similar heights between left and right subtrees.

Example:

AVL Tree

Condition:

|height(left) - height(right)| ≤ 1
